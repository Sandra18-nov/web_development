types of string methods

Python string methods functions
1.Capitalise -It makes the first letter uppercase
2.Upper -It makes all the letters uppercase
3.Lower -It makes all the letters lowercase
4.Isupper -It checks if all the letters are uppercase
5.Islower -It checks if all the letters are lower case
6.Isnumeric -It checks if all the characters are numeric
7.Isalpha -It checks if all the characters are alphanumeric
8.Isalnum -It checks if all the characters are alphanumeric
9.Count -It counts the number of occurences of the given character in a string
10.Find -It returns the index of the first occurence
11.Replace -It replaces a string or a part of it with the given set of characters
12.Split -It splits a string at the occurences of the specified character and returns a list that contains each part after splitting
